# 🤖 AI Calculator

An AI-inspired calculator built using **HTML, CSS, and JavaScript** with a modern glassmorphism interface and smart interactive features.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🌙 Dark / Light Mode
- 🎤 Voice Input
- 🤖 AI Assistant Messages
- 📋 Copy Result
- 📜 Calculation History
- 💾 Local Storage Support
- ⌨️ Keyboard Support
- 🗑️ Clear History
- 📱 Responsive Design
- ✨ Glassmorphism User Interface



## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Web Speech API
